# IM-Project-Vibration-Comfort-Estimation
Estimating the Vibration Comfort while operating Large Scale Shake Table Test Using AI Models, Deep Learning to create a surogate model for the same
This consists of:-
1. The Original Paper which was studies for the implementation and paper presenation for IM course
2. Paper Presentation PPT which contains the aim, basic understanding of the need, process of shake table tests, the advers efects and finally how the Deep Learning surrogate model is implemented using pre-processing od data, CNN model training and testing for the same.
3. Project Implementation PPT and word doc which has the simple implementation guidelines which were used for making the working model of the system using simple linear regression.
4. The source code in Python for the surrogate model made using Linear Regression.
